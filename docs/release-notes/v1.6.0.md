# v1.6.0 — AI & ML tools + bundled agent skills

**Date:** 2026-06-21

## Highlights

- **8 new AI/ML discovery tools** for Unity projects (no Editor required).
- **5 bundled Cursor-style agent skills** for ML-Agents, Sentis, NavMesh NPC, LLM integration, and AI stack audits.
- **110 tools** total.

## New tools

| Tool | Purpose |
|------|---------|
| `get_ai_ml_package_inventory` | ML-Agents, Sentis, Barracuda, AI Navigation, Muse, Unity AI packages |
| `list_ml_model_assets` | `.onnx`, `.nn`, `.pt`, `.tflite`, `.pb` under Assets |
| `find_ai_related_scripts` | Heuristic C# scan for AI/ML/NavMesh/LLM patterns |
| `list_ai_prompt_or_config_assets` | Prompt/RAG-like JSON or MD in Assets |
| `list_ml_agents_training_configs` | ML-Agents trainer YAML files |
| `get_ai_stack_summary` | One-shot audit of the full AI stack |
| `list_ai_skills` | List bundled Unity AI workflow skills |
| `read_ai_skill` | Load a skill by id |

## Bundled skills

Install path: `node_modules/unity-mcp-server/skills/<id>/SKILL.md`
