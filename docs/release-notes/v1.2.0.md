# Unity MCP Server Release v1.2.0

**Release date:** (see [CHANGELOG](../../CHANGELOG.md))

Large tool expansion: 28 new tools across physics, rendering, 2D, UI, Input, CI, and third-party integrations.

## Summary of changes

### Added

- **28 new tools**, including:
  - **Physics, pipelines, Timeline, 2D:** Physics settings, render pipelines, Timeline playables, 2D/tilemap assets.
  - **TMP & UI:** TextMeshPro fonts, TMP settings path, UI Toolkit (.uxml, .uss).
  - **Input:** Input System action assets and summary.
  - **Presets, editor scripts, prefab refs, assembly graph, CI:** Presets, editor scripts, prefab script GUIDs, assembly dependency graph, CI configs (.github/workflows, Jenkinsfile, etc.).
  - **Large assets:** List large assets by size.
  - **Integrations:** PlayFab, Figma, Firebase, Steam, Discord, FMOD, Wwise, Substance, SpeedTree, Lottie, analytics/crash (Sentry, Crashlytics, BugSnag), ads (Unity Ads, AdMob, ironSource).
  - **Version control:** Git LFS tracked patterns, Plastic SCM config.
